A configurable data-driven data table component for comparing items across multiple dimensions. Supports heatmap visualizations, row pinning, filtering, sorting, and customizable column configurations.

The controls below document options available to configure the component with your own data and settings.

See your changes live in the [Demo story](?path=/story/comparison-tool-comparisontoolcomponent--demo).

**Note:** This story is for testing visual configuration and layout only. Dynamic features like category dropdown selection, filtering, sorting, and pinning are implemented on the backend and will not update the displayed data in this demo. Heatmap popovers and view details links are also not enabled in this story.
